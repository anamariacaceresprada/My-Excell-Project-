# My-Excell-Project- Retail & Sales Analysis Project

## Overview
This project was completed as part of a **Data Technician Bootcamp** and focuses on analysing retail and sales data using **Microsoft Excel**. The goal was to explore, clean, and analyse transactional data to extract meaningful insights about sales performance and customer patterns.

The project demonstrates core spreadsheet and data analysis skills commonly used in entry-level data roles.

---

## Key Skills Demonstrated

### Excel Formulas
Used a range of formulas to transform and analyse the dataset:

- `SUM` – Calculate total sales and aggregated values  
- `SUMIF` – Conditional aggregation of sales by category or condition  
- `AVERAGE` – Determine average sales values  
- `AVERAGEIF` – Conditional averages for targeted analysis  
- `DATE`, `MONTH`, `YEAR` – Extract and analyse time-based information  
- `UNIQUE` – Identify unique categories or values in the dataset  
- `VLOOKUP` – Retrieve related data across tables  
- `SWITCH` – Categorise values and create readable classifications  

---

### Data Analysis Techniques
- **Filtering** to isolate specific categories or time periods  
- **Sorting** to organise sales values and identify top-performing items  
- **PivotTables** to summarise and explore large datasets quickly  
- **Data grouping** for time-based analysis (monthly/yearly trends)

---

### Data Visualisation
Created charts to clearly communicate insights, including:

- Sales trends over time
- Category-level sales comparisons
- Summary charts generated from PivotTable data

These visualisations help transform raw data into understandable insights for decision-making.

---

## Dataset
The analysis uses **retail and sales transaction data**, including fields such as:

- Product categories
- Sales amounts
- Transaction dates
- Customer and purchase information

---

## Learning Outcomes
Through this project I developed practical experience in:

- Analysing structured retail datasets
- Applying Excel formulas for real-world analysis
- Building PivotTables to summarise large datasets
- Creating clear visualisations for business insights
- Preparing data for reporting and decision support

---

## Tools Used
- **Microsoft Excel**
- PivotTables
- Excel Charts
- Data filtering and sorting tools

This work was completed during a **Data Technician Bootcamp**, where the focus was on building foundational data analysis skills using industry-standard tools.
